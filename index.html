Pengertian Hardware Komputer dan Laptop

Hardware adalah bagian fisik dari komputer atau laptop yang bisa dilihat dan disentuh secara langsung. Komponen hardware berperan penting dalam menjalankan seluruh fungsi komputer, mulai dari proses input, pengolahan data, hingga output.

Gambar: Komponen Hardware Komputer

Pengertian Komponen Hardware

1. PSU (Power Supply Unit)

PSU adalah perangkat yang mengubah arus listrik dari sumber (PLN) menjadi arus yang sesuai untuk komponen komputer lainnya.

2. Mainboard / Motherboard

Mainboard adalah papan sirkuit utama tempat semua komponen komputer terhubung.

3. GPU (Graphics Processing Unit)

GPU memproses tampilan visual dan sangat penting untuk grafis, video, dan game.

4. Processor (CPU)

CPU adalah otak komputer, menjalankan semua perintah dan pemrosesan data.

5. Sound Card

Mengatur input dan output suara dari komputer ke speaker, headphone, dan mikrofon.

6. Mouse

Perangkat input untuk mengontrol kursor dan menavigasi antarmuka pengguna.

7. Keyboard

Perangkat input untuk mengetik teks, angka, dan menjalankan perintah tertentu.

8. Printer

Mencetak dokumen atau gambar dari komputer ke media fisik seperti kertas.
